---
title: 面向切面编程AOP
---

用途： 业务中降低耦合性的有效手段之一

应用： Spring 框架的核心功能

### OOP VS AOP

|      | OOP              | AOP                                    |
| ---- | ---------------- | -------------------------------------- |
| 关注 | 对象内在的自洽性 | 系统本身的行为                         |
| 适合 | 业务功能         | 统一的行为动作                         |
| 案例 | 商品、订单、会员 | 日志记录、性能统计、权限控制、事务处理 |
