### Map.values转数组
Array.from (推荐)
```typescript
const map1 = new Map().set('0', 'foo').set(1, 'bar');
console.log(Array.from(map1.values()));
// Array ["foo", "bar"]
```
[...] (不推荐，ts中编译告警)
```typescript
const map1 = new Map().set('0', 'foo').set(1, 'bar');
console.log([...map1.values()]);
// Array ["foo", "bar"]
```
